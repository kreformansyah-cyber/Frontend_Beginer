# Frontend Beginner Project

This repository contains a beginner-level frontend project created by following
the **Frontend Beginner Roadmap** from [roadmap.sh](https://roadmap.sh/frontend).

The goal of this project is to build a strong foundation in **HTML**, understand
**semantic markup**, basic **SEO**, and prepare for the next steps such as CSS
and JavaScript.
